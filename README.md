# 4D Studio

A lightweight, browser-based 3D animation studio. No installs, no subscriptions — just open the HTML file and start animating.

Built with [Three.js](https://threejs.org/). The entire studio is a single HTML file under 50KB.

![4D Studio Screenshot](screenshot.png)

## Features

### Scene Management
- **Import GLB/GLTF models** — drag & drop or click to import
- **Import animation JSON** — load AI-generated or hand-crafted animation data
- **Export GLB** — save your scene as a single file
- **Export JSON** — save positions, animations, and keyframe data
- **Multi-model support** — load multiple models, position them together

### Posing & Animation
- **Bone selection** — click bones in the viewport or scene tree
- **Transform gizmo** — move, rotate, scale with visual handles (W/R/S keys)
- **Animation timeline** — add keyframes, scrub time, play/pause
- **Draggable keyframes** — slide keyframe diamonds to adjust timing
- **Load existing clips** — click any animation clip to load it into the timeline for editing
- **Save animations** — save your work as new clips on the model
- **Channel locking** — lock channels to protect finished animation while editing others
- **Lock All / Unlock All** — quick buttons for bulk locking

### Visualization
- **Outline highlighting** — hover to see amber outline around parts (toggle with H)
- **Chain mode** — see connected bone hierarchy highlighted in cyan (toggle with C)
- **Skeleton visualization** — bone dots and skeleton lines (toggle in toolbar)
- **Hover labels** — see bone/part names on hover

### Editing
- **Undo / Redo** — Ctrl+Z / Ctrl+Shift+Z (50 levels)
- **Right-click rename** — rename models, bones, and animations
- **Snap mode** — snap to grid for precise positioning
- **Collapsible panels** — collapse tree/properties panels for more viewport space
- **Resizable timeline** — drag the top edge to resize

## Getting Started

1. Download `index.html`
2. Open it in Chrome, Firefox, or Edge
3. Drag a `.glb` file onto the viewport
4. Click bones to select them, use W/R/S to transform
5. Press K to add keyframes, Space to play

That's it. No build step, no npm install, no server required.

## AI Animation (WIP)

The studio supports importing JSON animation data generated by AI. The pipeline:

1. Export your model's bone data from the studio
2. Describe the animation you want (walk cycle, wave, throw, etc.)
3. AI generates keyframe JSON with proper bone rotations
4. Import the JSON — it becomes a playable clip

This feature is a work in progress. Contributions welcome to improve bone orientation detection and animation quality.

## Tech Stack

- **Three.js** (r160) — 3D rendering, loaded from CDN
- **PostProcessing** — OutlinePass for selection/hover highlighting
- **Pure HTML/CSS/JS** — no framework, no build tools, no dependencies

The entire studio runs in the browser using WebGL. Your GPU does the heavy lifting.

## Why?

Blender is 3+ million lines of code and a 2GB download. Most game devs using it for animation touch maybe 5% of its features.

This studio does exactly what you need: load models, move bones, set keyframes, export. In a single file you can email.
